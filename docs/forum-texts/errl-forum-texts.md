## Usage Notes

### Mixing and Matching
- **Start simple:** Use the "Best Fit" set for initial implementation
- **Add personality:** Swap in creative variations based on context
- **Lore depth:** Use lore-inspired text for dedicated Errl community spaces
- **Accessibility:** Keep core functionality clear, add flair around edges

### When to Use What
- **Subtle themed set:** Professional but playful (good for public-facing)
- **Chaotic themed set:** Maximum personality (good for internal/community)
- **Lore-inspired:** Deep community engagement (good for dedicated fans)
- **Creative variations:** Context-aware experiences (time, mood, activity)

### Tone Consistency
- Keep it friendly, never mean
- Weird is good, confusing is not
- Short beats long
- Personality beats generic
